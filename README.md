# Project Description

This web app has been developed for the IOT club of my college.
If is fully functional with both user and admin dashboard




## Features

- Fully responsive
- User registration 
- User login/logout system with session  
- Forget password system  
- Emailing system 
- User dashboard
- Admin dashboard
